图像的参数​

像素:组成图像的最小单元 像素分辨率:每英寸储存的像素数 像素深度:存储每个像素所用的数据位数,常用:8bit,10bit,12bit,16bit 24bit用于彩色 灰度：色阶/灰阶,亮暗程度，数值越大，越明亮

  • 黑色:0
  • 白色:255 颜色模型
  • RGB(红，绿，蓝)
  • HSB(色度，饱和度，亮度)
  • CMYK(青色，洋红色，黄色，黑色)
  • Lab(亮度，从洋红色到绿色的范围，从黄色到蓝色的范围)

图像噪点:图片中不必要的干扰信息,形成原因可能就是感光面积小，转换成电信号的效率比较低

工业相机的主要参数​

1. 分辨率​

像素点按照矩阵的方式进行排列

  • 面阵相机:水平方向*垂直方向的像素数
  • 线阵相机:传感器方向上的像素数

2. 芯片尺寸​

芯片尺寸越大,对噪点的控制越好

3. 曝光​

正确曝光->控制曝光量

  • 感光度(ISO值) ISO变大:提高曝光程度,增加噪点
  • 快门(电子快门)
  • 光圈
