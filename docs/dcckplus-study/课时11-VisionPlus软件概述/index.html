VisonPlus软件概述

V+初识​

DCCK VisionPlus平台软件是一款集开发、调试和运行于一体的可视化的机器视觉解决方案集成开发环境，无代码编程。专注于机器视觉应用，集成了采集通讯、视觉算法、数据分析、行业模块、人机交互以及二次开发等视觉项目常用功能和模块。

V+平台功能框图​

V+软件功能特点​

  1. 软件提供可视化开发环境，以"所见即所得"的设计理念，支持用户以拖、拉、选、填的方式构建项目流程，无需编程即可快速开发完整的视觉应用程序。
  2. 软件集成兼容了国内外多种品牌的硬件设备与通讯协议，如相机、光源、3D相机、PLC，IO卡等。系统层面提供统一操作方式，用户无需关注底层驱动。简单设置即可便捷操作。
  3. 软件开放HMI界面的可视化编辑功能，支持用户以"所见即所得"的方式自主设计软件交互界面，充分满足用户对效率、自主性、美观等方面的需求。
  4. 软件集成康耐视VisionPro视觉算法库，深度融合康耐视高精度算法，工具齐全，稳定可靠,可满足2D和3D视觉应用领域的定位、测量、检测、识别等视觉任务要求。
  5. 软件底层采用多线程并行运行机制，无需用户参与系统会自动识别可并行的流程，高效运行。
  6. 软件提供典型视觉应用场景行业模块与项目模板，目前已支持连接器、引导、检测等领域。通过将高级工程师的经验集成在模块中，为行业用户赋能，大幅降低开发门槛并提升效率。
  7. 软件在兼容多种设备并提供了丰富的工具的同时，也具备高的可扩展性，还提供了表达式、脚本和二次开发功能，用户可以很方便的实现自定义设备、工具以及项目特定逻辑功能。
