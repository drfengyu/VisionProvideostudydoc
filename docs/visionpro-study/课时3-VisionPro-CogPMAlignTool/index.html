PMAlign-1

图像搜索 定位 主要功能:根据图像当前的模板对图像进行识别和定位

CogPMAlignTool的操作方法​

  1. 抓取训练图像
  2. 拖动区域框调整大小
  3. 中心原点
  4. 训练
  5. 运行参数 设置角度 匹配不同角度图片

CogPMAlignTool的基本原理​

  • 基于图像轮廓进行搜索定位的工具 也称边缘特征
  • 图形选项卡点击显示精细 显示粗糙
  • 离散的小线段 绿色线比黄色线要小 但多
  • 边缘是一个有大小和方向的矢量
  1. 点击浮动显示 查看训练图像 可以看到右侧灰度值是77 左侧白色区域是180左右 幅度大概是100 中间区域左右幅度差值大概是30
  2. 具有方向性 由明到暗 由暗到明
  3. 边缘特征越多越精细 准确度越高
  4. 边缘特征越大越少 匹配速度也越快 但准确度也会变差

CogPMAlignTool的训练参数介绍​

  1. 训练参数 展开 特征粒度限制 自动选择 会默认自动计算出一个粗糙和精细的比例值
  2. 特征粒度限制:粗糙>=精细 如果精细设为7 粗糙大于等于7 颗粒度越大 特征选取的越少
  3. 像素可以达到1/40个像素
  4. PatMax定位精度会比PatQuick要高，速度没PatQuick快,PatMax与PatQuick综合,PatFlex训练图像是平整的,使用于发生一些畸变的情况
  5. 极性 帮助文档 UserGuide-Search-PMAlign-PatMax Software
  6. 弹性 默认使用0 调大 会影响匹配的准确程度 允许有一定的程度
  7. 自动边缘阈值 调大 会忽略掉部分特征

课程小结​

  1. CogPMAlignTool的操作方法 抓取图像-设置训练区域及参数-训练模板-设置运行参数与区域-运行-查看结果
  2. CogPMAlignTool的基本原理
  • 图案位置搜索工具(识别+定位)
  • 基于边缘特征的模板而不是基于像素的模板匹配
  • 支持图像中特征的旋转与缩放
  • 边缘特征表示图像中不同区域间界限的轮廓线 (有大小和方向)
  1. CogPMAlignTool参数介绍
  • 训练参数 : 算法、忽略极性、粗糙度、精细度、边缘阑值、弹性。
  • 保存、加载模板
